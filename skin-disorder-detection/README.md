# Detecting the Type of Skin Disorder

## Objective
Build a machine learning model to classify 6 categories of skin disorders using 
clinical and dermatological features.

## Approach
- Data preprocessing and handling of missing values (e.g. Age feature)
- EDA including correlation heatmap and feature-vs-target analysis
- Label encoding and feature scaling
- Compared 8 classification models (Logistic Regression, Decision Tree, Random 
  Forest, Gradient Boosting, XGBoost, SVM, KNN, Naive Bayes)
- Evaluated using accuracy, confusion matrix, and feature importance

## Result
Best Model: **Gradient Boosting** — Accuracy: **98.6%+**
Grade received: B

## Tools
Python, scikit-learn, XGBoost, Matplotlib, Seaborn
